       - `[[env.{client}.d1_databases]]` with binding=DB, database_name, database_id
       - `[[env.{client}.r2_buckets]]` with binding=MEDIA_BUCKET, bucket_name
       - `[[env.{client}.kv_namespaces]]` with binding=CACHE_KV, id
    9. Print next-steps:

       - `wrangler secret put JWT_SECRET --env {client}`
       - `wrangler secret put WEBHOOK_SECRET --env {client}` (optional)
       - `wrangler deploy --env {client}`
       - Seed admin user (reference seed-admin.ts or manual SQL insert)
